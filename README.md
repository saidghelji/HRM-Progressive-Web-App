# HRM Progressive Web App

A modern HR management dashboard prototype built as a progressive web app with offline capabilities, employee workflows, and advanced HR tooling.

## 🚀 Project Overview

This HRM app simulates an enterprise HR system with:
- Employee management
- Time-off workflow
- Candidate tracking
- Performance and analytics dashboard
- Offline support via PWA

## 🧩 Features

- Employee CRUD management
- Time-off requests and approval workflow
- Candidate hiring pipeline
- Goals tracking and analytics
- Real-time notifications
- Offline-first support with service workers

## 🛠️ Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- PWA
- Service Workers
- Local Storage
